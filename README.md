# Ideasphere Website

A modern, professional website for Ideasphere Knowledge Partner Pvt Ltd - a Venture Transformation Studio helping build next-generation, impact-driven businesses.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI**: React 19

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
├── app/                  # Next.js App Router pages
│   ├── page.tsx         # Home page
│   ├── about/           # About page
│   ├── services/        # Services page
│   ├── products/        # Products page
│   ├── how-we-work/     # How We Work page
│   ├── impact/          # Case Studies page
│   ├── careers/         # Careers page
│   └── contact/         # Contact page
├── components/          # Reusable React components
│   ├── layout/         # Layout components (Navbar, Footer)
│   └── ui/             # UI components
├── data/               # Dummy data files
└── public/             # Static assets
```

## Pages

- **Home**: Hero section, services overview, products showcase, process flow
- **About**: Company information and mission
- **Services**: Detailed capabilities and offerings
- **Products**: Portfolio of internal products and platforms
- **How We Work**: 5-step methodology visualization
- **Impact**: Case studies and success stories
- **Careers**: Job openings and internship programs
- **Contact**: Contact form and company details
